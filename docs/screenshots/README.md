# Screenshot Placeholders

Add portfolio screenshots here when available. Recommended images:

- `dashboard.png` - current lean-bulk status.
- `daily-entry.png` - daily tracking entry form.
- `progress.png` - progress chart or trend view.
- `recommendation.png` - calorie recommendation state.
- `mobile-pwa.png` - mobile responsive/PWA view.

Do not commit screenshots that expose private bodyweight data, personal notes, Firebase project secrets, or local-only credentials.
